## ✨ Features / 特徴

- **Li (Limit & Link)**: 厳格なアクセス制御。回数制限や有効期限を1秒単位で設定可能。
- **tr (Track)**: リアルタイムなクリック解析。誰がいつアクセスしたかを透明化。
- **Anonymous**: ユーザー登録不要。秘密の「管理URL」だけで全ての操作が可能。
- **Aurora UI**: ガラスモーフィズムとオーロラ背景を採用した、没入感のあるデザイン。
- **i18n Ready**: 日本語・英語に完全対応。

## 🚀 Quick Start / 使い方

1. `index.html` から転送先URLを入力
2. 必要に応じて上限回数や期限を設定
3. **Short URL** と **Admin URL** を取得
4. 管理用URLをブックマークしておけば、後からいつでも統計確認や設定変更が可能！
